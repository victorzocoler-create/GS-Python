O Helpjob é um aplicativo criado para auxiliar pessoas a entenderem como diferentes profissões podem ser impactadas pela automação e pela evolução das tecnologias emergentes. Por meio de perguntas guiadas, o sistema identifica o grau de automatização de uma profissão, apresenta possíveis áreas de trabalho na mesma área e recomenda profissões do futuro de acordo com os interesses do usuário.


# Funcionalidades Principais

* Determina o grau de automação de uma profissão (Alta, Média ou Baixa).
* Explica detalhes sobre a profissão e como ela pode ser impactada.
* Sugere profissões do futuro com base na área de atuação.
* Permite que o usuário crie uma lista personalizada de profissões de interesse.



# Estrutura do Código

O projeto está dividido em funções principais:

* **`interface()`** – Exibe o texto de boas-vindas e orienta o usuário.
* **`limpar_texto()`** – Limpa o texto para melhor experiencia do usuario
* **`descricao_emprego()`** – Explica o impacto da automação na profissão.
* **`principal()`** – Lógica principal de análise das profissões.
* **`ajuda()`** – Fluxo de sugestões de profissões do futuro.



# Dependências

O programa é simples e construído para funcionar sem dependências externas.



# Exemplo de Uso

1. O usuário digita a profissão desejada.
2. O sistema informa o nível de automação.
3. O usuário decide se quer mais detalhes.
4. O sistema oferece sugestões de profissões do futuro.
5. O usuário monta sua própria lista de interesse.
